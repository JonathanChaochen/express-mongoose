# Node.js Express Mongoose

### `Install`
 
Run 
```sh 
npm install
```

Run 
```sh
npm run dev
```

The server will run on port 3000

### `Dependencies`

```js
  "dependencies": {
    "body-parser": "^1.18.2",
    "express": "^4.16.3",
    "mongoose": "^5.0.15",
    "morgan": "^1.9.0"
  },
  "devDependencies": {
    "nodemon": "^1.17.3"
  }
```

### `MongoDB setup `
Mongoose is a MongoDB object modeling tool designed to work in an asynchronous environment.
